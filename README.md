
# <p align="center"><b>GameBoy</b></p>
### <p align="center"><b>Simple code to make Old School style GameBoy using Attiny85</b></p>

## Available Games
- Dino Game
- Flappy Bird

  - Will be added
  - Tic tac toe
  - Pong

## Needed Components
- Arduino IDE App
- Attiny85 Chip
- Oled Display 0.96"
- 3x arduino Buttons
- 3X Resistor with 10K Ohm
- Arduino Breadboard or Universal PCB
